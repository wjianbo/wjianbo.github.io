我基于青空文库的 GitHub 仓库，开始了一个新项目。

它只是[一个很简单的网页](/aozorabunko)，每天展示一句来自青空文库的文字。

  

这是我欠自己的东西。

  

我发现，只要脑子里冒出一个想法，我就会隐约觉得自己对它负有责任。如果不把它真正做出来，我会担心它慢慢溜走。不是因为这个想法不够好，而是因为我没有在它出现的时候，认真对待它。

  

我卧室里的 Wi-Fi 信号很弱，iPhone 也经常断网。

但我并不介意。某种程度上，断开连接反而让人安心。如果有什么事情必须在线完成，明天再做也来得及。

  

正是在这种状态下，我又想到了一个新点子。

  

我想做一个 Chrome 扩展，名字叫「Aozora Audio」。

当青空文库的文章存在音频资源时，扩展会在页面上显示一个播放按钮。用户不需要额外查找，只要点一下，就可以直接收听。
