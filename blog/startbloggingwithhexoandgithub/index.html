Hexo+Github 搭建博客（Mac）

安装

  1. 安装 Hexo 前需要先安装下列应用程序：
  • Node.js
  • Git
  1. 安装 Hexo
$ npm install -g hexo-cli

建站

$ hexo init <folder>
$ cd <folder>

配置

在 _config.yml 中修改配置。

更换主题

换成我比较喜欢的 next 主题。

  1. 安装
$ git clone https://github.com/iissnan/hexo-theme-next themes/next
  1. 修改配置文件（_config.yml） 
theme: next
  1. 链接失效问题 问题描述：「首页」「归档」等链接失效，URL 地址最后面出现「%20」。 解决方法：去掉主题配置文件（theme/next/ _config.yml）里menu部分多余的空格，例如：
home: / || home

改为：

home: /|| home

预览

$ hexo server

启动服务器。默认情况下，访问网址为：http://localhost:4000/

部署

  1. 安装 hexo-deployer-git 
$ npm install hexo-deployer-git —save
  1. 修改配置（_config.yml） 
deploy:
  type: git
  repository: https://github.com/<你的 GitHub 用户名>/<你的 GitHub 用户名>.github.io.git
  branch: master
  1. 生成站点文件并推送至远程库。
$ hexo clean && hexo deploy

写作

$ hexo new [layout] <title>
$ hexo clean && hexo deploy
