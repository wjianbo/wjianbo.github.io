用 Git 管理博客

昨天安装了 Hugo，但还只能在本地运行，没有上传到 GitHub。今天一早起来把源码和静态文件放在同一个 Repo 中。生成的静态文件放在 docs 文件夹中。因为在 Pages 设置中可以指定该文件夹。修改后，生成静态文件时需指定文件夹：

hugo -d docs -D

现在，本博客所有源文件都放在 GitHub 上了。
